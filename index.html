Simon Game

A replication of the Simon game built using HTML5, CSS3, and Javascript. I built this to experiment with temporarily storing data in Javascript arrays. The website randomly generates a new color for you to remember each round, stores that data into its array, and listens to the user input as they try to click all the colors that have been selected so far. If they are successful, it starts a new round. If not, they are invited to start a new game. This combines the interactivity of my drum kit with the Javascript math functions from my Dice Simulator as well as a new thing, arrays, to make a fun and challenging game.

To-Do List

Web application built using primarily HTML5, CSS3, Node.js, Express.js, MongoDB, and Mongoose. This is one of the most complex projects I've worked on, although on the surface it appears simple. I knew how to make a website interactive, but I wanted to take it to another level and actually store the data a user submits to and removes from a website, so that when they come back later or refresh the page it's still exactly as they left it. This required learning around 4 new programming languages, some for running the server which listens to input from the user and runs the right code for that interaction and others for databases to store the data within the webpage and keep it safe after the user leaves.
